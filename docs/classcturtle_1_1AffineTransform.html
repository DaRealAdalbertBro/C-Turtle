Detailed Description

The AffineTransform class provides a myriad of functions to simply transform points. This class it the backbone of almost all cartesian plane math in CTurtle. An adapted 3x3 matrix of the following link: http://www.opengl-tutorial.org/beginners-tutorials/tutorial-3-matrices/.
